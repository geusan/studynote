    - [ 제어판 ] - [ 네트워크 및 공유센터 ] - [ 어댑터 설정 ]
    - WIFI는 [ 속성 ] - [ 공유 ] - 다른네트워크가 사용설정 체크

    // /etc/dhcpcd.conf 파일

    profile static_eth0

    static routers=192.168.1.1
    static domain_name_servers=192.168.137.1

    # fallback to static profile on static_eth0
    interface static_eth0
    fallback static_eth0
